Download lumberjack

A device library and logging system using RFID technology.

lumberjack is an open source project, so you can download it from our GitHub.

How to run lumberjack

Create a mySQL database, and change the configuration in the code to connect to your local/remote database.

To run the project, navigate to the root of the project directory lumberjack_spring and run Spring Boot mvn spring-boot:run.

Navigate to localhost:8080 enter the web front-end.
